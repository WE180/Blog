---
title: vuepress-theme-reco 安装
date: 2021-04-27 00:49
tags:
 - vuepress
---

**npx**

```bash
npx @vuepress-reco/theme-cli init
```

**npm**

```bash
# 初始化
npm install @vuepress-reco/theme-cli -g
theme-cli init
```

**yarn**

```bash
# 初始化
yarn global add @vuepress-reco/theme-cli
theme-cli init
```
